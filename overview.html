j(s)-lib Tools - Extension Tasks for Apache Ant

This project contains Ant tasks for building j(s)-lib libraries and application based on them.

Tasks Usage

There are two basic methods to execute user defined tasks. Standard Ant task usage pattern to declare task definition using taskdef task, as in snippet below. 

<taskdef name="js.BuildResources" classname="js.tools.ant.task.BuildResources">
	<classpath>
		<pathelement path="${LIB-JS-ANT-TASK}" />
		<pathelement path="${LIB-JS-CLIENT}" />
		<pathelement path="${LIB-JS-DOM}" />
		<pathelement path="${LIB-WOOD}" />
		<pathelement path="${LIB-RHINO}" />
		<pathelement path="${LIB-XERCESIMPL}" />
	</classpath>
</taskdef>
. . .
<target name="build resources">
	<js.BuildResources dir="build/site" verbose="true" />
</target>

If using Eclipse IDE one can declare task in Window/Preferences/Ant/Runtime/Tasks and use it directly by name. 

<target name="build resources">
	<js.buildResources dir="build/site" verbose="yes" />
</target>
